ผลงาน (Portfolio)

📚 ด้านวิชาการ

  • เข้าร่วมการแข่งขัน/โครงงานวิทยาศาสตร์
  • เข้าร่วมการแข่งขันทักษะคณิตศาสตร์
  • ได้รับเกียรติบัตรด้านผลการเรียนหรือการแข่งขันทางวิชาการ

💻 ด้านทักษะและความสามารถพิเศษ

  • เขียนโปรแกรม HTML, CSS และ JavaScript
  • ใช้โปรแกรม Microsoft Office
  • ออกแบบสื่อด้วย Canva
  • มีทักษะการนำเสนอและการทำงานเป็นทีม

👨‍💼 ด้านกิจกรรมและภาวะผู้นำ

  • เข้าร่วมกิจกรรมของโรงเรียน/มหาวิทยาลัย
  • เป็นหัวหน้ากลุ่มในการทำโครงงาน
  • ร่วมวางแผนและดำเนินกิจกรรมต่าง ๆ

❤️ ด้านจิตอาสาและการบำเพ็ญประโยชน์

  • เข้าร่วมกิจกรรมจิตอาสาพัฒนาชุมชน
  • ร่วมทำความสะอาดโรงเรียนและพื้นที่สาธารณะ
  • ปลูกต้นไม้และอนุรักษ์สิ่งแวดล้อม

💼 ด้านการทำงานหรือฝึกงาน

  • ฝึกงานด้านเทคโนโลยีสารสนเทศ
  • ฝึกการทำงานร่วมกับทีมและแก้ไขปัญหา
  • เรียนรู้การใช้ระบบงานจริงในสถานประกอบการ

🎓 ด้านการอบรมและสัมมนา

  • อบรมการเขียนโปรแกรมเบื้องต้น
  • อบรมการใช้เทคโนโลยีดิจิทัล
  • เข้าร่วมสัมมนาด้านวิทยาศาสตร์และนวัตกรรม
